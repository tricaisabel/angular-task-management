Welcome to Task Overflow!

Hello everyone! This is a practice app developed by Isabel Trica on her to learn Angular, Nest.js and Postgresql.

What you can do

  • Create, sort and filter boards
  • Add, assign and edit tasks
  • Any user can create their own boards that they can later manage
  • Tasks can have one of the following types: Bug, User Story, Issue
  • Track progress. The board can be visualised in the ListView or Kanban mode

Technical Features

  • Lazy loading - for fast rendering of the page
  • Route protection - agains unauthorized users
  • File Management - all images are stored safely on a server
  • Auto-login - users stay logged in when refreshing the page
  • Auto-logout - users are automatically logged out after 1 hour
  • Drag and Drop Cdk - used for the Kanban Board

Welcome to Task Overflow

Hello everyone! This is a practice app developed by Isabel Trica on her journey to learn Angular, Nest.js and Postgresql.


  • Create, sort and filter boards
  • Any user can create their own boards that they can later manage
  • Add, assign and edit tasks
  • Tasks can have one of the following types: Bug, User Story, Issue
  • Track progress. The board can be visualised in the ListView or Kanban mode
